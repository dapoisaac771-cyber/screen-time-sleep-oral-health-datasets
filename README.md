# Dataset: Impact of Screen Time and Sleep Quality on Oral Behaviors and Caries Risk in School-Age Children

**Author:** Isaac  
**Corresponding author email:** dapoisaac771@gmail.com

## Description
This repository contains the dataset and analysis materials used in the study:
"Impact of Screen Time and Sleep Quality on Oral Behaviors and Caries Risk in School-Age Children".

Files:
- `turkish_children_oral_health_data.csv`
- `turkish_oral_health.pdf`
- `turkish_oral_health_syntaxfile.docx`
- `Sleep_quality_research.docx`
- `tables_and_figures/` (figures for the paper)
- `scripts/` (SPSS syntax and analysis files)
